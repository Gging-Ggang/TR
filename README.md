# 턴제 전투 게임 (TR Project)

간단하지만 전략적인 메커니즘을 가진 캐릭터들이 대결하는 턴제 전투 시뮬레이터입니다.

## 🚀 실행 방법

### 1. 웹 UI (브라우저 플레이)
- `index.html` 파일을 브라우저로 엽니다.
- 실시간 로그, 스택 기반 게이지 효과, 상태 이상 시각 효과(방어 방패, 침묵 등)가 제공됩니다.

---

## 🎮 캐릭터 상세 설명

### 1. 쿠키 (흑마도사)
**체력: 70**

**패시브: [천상의 화염]**
- **개전**: 게임 시작 시 [도화선]을 5 획득합니다.
- **반복**: 턴 시작 시 1d3(강화 시 1d4)을 굴려 **[불꽃]**을 얻습니다.
    - **[도화선]** 및 **[역화]** 획득: 얻은 [불꽃]만큼 수치가 증가합니다.
    - **보호막** 획득: `(주사위 최대값 - [불꽃]) * 4` 만큼의 보호막을 얻습니다.
- **피격**: 보호막이 아닌 체력이 닳았을 때 1d3만큼 [도화선]이 감소합니다.

**일반 공격: 마법 화살**
- [불꽃]d6 만큼 피해를 입힙니다.

**액티브 1: 하이 파이쟈 (쿨타임 1턴 + [불꽃])**
- **[도화선]d[불꽃]** 만큼의 피해를 주고 [도화선]이 절반으로 감소합니다.

**액티브 2: 불덩이 작렬 (쿨타임 3턴)**
- `(1d3 - 1)` 피해를 **[도화선]** 횟수만큼 입힙니다. 0 피해 발생 시 도화선과 역화가 충전됩니다.

**액티브 3: 마력 각성 (쿨타임 4턴)**
- 사용 시 현재 걸린 **[방어] 효과를 즉시 제거**하고, **방어 스킬의 쿨타임을 1턴 증가**시킵니다.
- 15의 보호막을 얻으며, 다음 턴 주사위를 두 번 굴려 패시브 효율을 극대화합니다.

**궁극기: 역화 폭발**
- **사용 조건**: [역화] 30 이상.
- **영창 (2턴)**: 모든 역화를 소모하여 **[역화 보호막]**을 생성합니다. 영창 중에는 패시브가 중단됩니다.
- **발동**: 공격 직전 **패시브가 다시 발동**하며, **[일반 피해 30 + 남은 역화 보호막만큼의 고정 피해]**를 입힙니다.

### 2. 솔리스 (암살자)
**체력: 90**

**패시브: 계획**
- **계획**: 턴 시작 시 1d3 주사위를 굴립니다.
    - **1**: 이번 턴 상대의 모든 공격을 **회피**합니다. (이로운 효과)
    - **2**: 모든 스킬의 쿨타임이 1턴 감소합니다.
    - **3**: **'살의'** 1스택을 획득합니다.
- **공백**: 모든 스킬이 쿨타임일 경우 보호막 4를 획득합니다.
- **살의**: 일반 공격 시 1스택 획득. **10스택** 소모 시 공격에 **[대상 최대 체력의 10% 고정 피해]**를 추가합니다.

**액티브 1: 속임수 공격 (쿨타임 3턴)**
- 2d6+4 피해 및 살의 2스택 획득. 다음 공격 시 2d6 추가 피해를 입힙니다.

**액티브 2: 비열한 기습 (쿨타임 5턴)**
- 2d6 피해를 입히고 상대의 모든 **[이로운 효과]**를 제거합니다.

**액티브 3: 몽환삼단 (쿨타임 6턴)**
- (2d3+3) x 3회 피해를 입히고 대상에게 2턴간 **[침묵]**을 부여합니다.
    - **침묵**: 일반 공격과 방어만 가능합니다. (해로운 효과)

**궁극기: 결심 (쿨타임 8턴)**
- **6턴부터 사용 가능**. [결심] 2스택 및 살의 1스택을 획득합니다.
- **결심 상태**: 공격 시 스택을 소모하여 **치명타(1.5배)** 적용, 살의 1스택 추가 획득, 대상이 주는 피해를 1턴간 **25% 감소**시킵니다.

---

### 3. 아모르 (팔라딘)
**체력: 100**

**패시브: 페르소나**
- 받은 피해가 44 누적 시 4턴간 **'어둠'** 상태가 되어 스킬이 강화되고 쿨타임이 초기화됩니다.

**액티브 1: 자활 / 집념 (쿨타임 2턴)**
- **빛(자활)**: 피해량의 절반만큼 보호막을 얻으며, 파괴 시 체력을 회복합니다.
- **어둠(집념)**: 더욱 강력한 추가 피해를 입힙니다.

**액티브 2: 백야 / 흑야 (쿨타임 4턴)**
- 보호막 8을 얻고 다음 턴까지 피격 시 생명력을 회복(빛)하거나 피해를 반사(어둠)합니다.

**액티브 3: 혼돈선 (쿨타임 7턴)**
- 3턴간 피해를 반감하고 체력을 재생(빛)하거나, 받은 피해를 축적하여 해제 시 방출(어둠)합니다.

---

## ⚖️ 전투 매커니즘

- **방어**: 쿨타임 3턴. 1턴간 받는 **일반 피해를 90% 감소**시킵니다. (이로운 효과가 아니므로 해제되지 않음)
- **피해 유형**:
    - **일반 피해**: 방어 효과와 보호막에 의해 경감됩니다.
    - **고정 피해**: 방어 효과를 무시하지만, **보호막으로 경감 가능**합니다.
- **상태 이상**:
    - **이로운 효과 / 해로운 효과**: 각 효과는 명시된 타입에 따라 해제 기술의 영향을 받습니다.
    - **침묵**: 스킬 사용을 봉쇄하며 오직 일반 공격과 방어만 허용합니다.
